## ⚔️ 核心设计原则
1. **完全还原技能效果**：所有技能的伤害、治疗、冷却时间、效果都和WLK版本一致
2. **天赋树还原**：每个职业3系天赋，每系至少还原20个核心天赋点
3. **职业特色保留**：
   - 战士：怒气系统，姿态切换
   - 圣骑士：圣能系统，光环效果
   - 死亡骑士：符文系统，符文能量
   - 猎人：集中值系统，宠物系统（保留宠物，取消随从系统）
   - 萨满：图腾系统
   - 盗贼：能量系统，连击点
   - 德鲁伊：形态切换（熊/猫/树/鸟）
   - 牧师：双天赋玩法（戒律/神圣/暗影）
   - 法师：法术控制，传送门
   - 术士：灵魂碎片，宠物系统

4. **数值平衡**：参考WLK版本官方数值，适当调整适配回合制玩法

## 🎯 技能系统设计
### 技能分类
1. **基础技能**：职业通用技能，1级即可学习
2. **职业技能**：各职业特有技能，随等级解锁
3. **天赋技能**：点天赋后解锁的强力技能
4. **专业技能**：专业相关技能（挖矿、采药等）

### 战斗属性
| 属性 | 作用 |
|------|------|
| 力量 | 增加物理攻击强度，板甲职业护甲加成 |
| 敏捷 | 增加物理暴击，攻击速度，盗贼/德鲁伊猫形态AP |
| 智力 | 增加法术强度，法术暴击，法力值上限 |
| 耐力 | 增加生命值上限 |
| 精神 | 增加法力回复速度，治疗效果加成 |
| 命中 | 降低攻击被闪避/招架的概率 |
| 暴击 | 攻击/治疗暴击概率 |
| 急速 | 降低技能冷却时间，增加攻击速度 |
| 精通 | 各职业特有加成效果 |
| 护甲 | 降低受到的物理伤害 |
| 魔法抗性 | 降低受到的对应系魔法伤害 |

## 🌲 天赋树设计
每个职业3系天赋，每系天赋树包含：
- 5层天赋，每层1-3个可选天赋
- 底层核心天赋需要至少投入31点对应系天赋点才能点出
- 天赋点总数：80级共71点天赋点
- 支持双天赋切换，可随时切换两套天赋配置

### 天赋示例（战士 - 武器系）
| 层数 | 天赋 | 效果 |
|------|------|------|
| 1 | 偏斜 | 增加5%招架概率 |
| 1 | 强化英勇打击 | 英勇打击怒气消耗降低3点 |
| 2 | 钢铁意志 | 增加10%抵抗昏迷/恐惧概率 |
| 2 | 强化撕裂 | 撕裂伤害提高20% |
| 3 | 重伤 | 暴击后使目标流血，6秒内受到相当于武器伤害48%的伤害 |
| 3 | 强化压制 | 压制伤害提高20% |
| ... | ... | ... |
| 底层 | 致死打击 | 强力攻击，造成武器伤害+额外伤害，并使目标受到的治疗效果降低50%，持续10秒 |

## 🔄 回合制适配调整
1. **公共冷却时间**：统一为1回合，技能冷却按回合数计算
2. **资源系统**：
   - 怒气：受到伤害或造成伤害时获得，每回合自动衰减
   - 法力值：施法消耗，每回合自动回复部分
   - 能量：每回合回复固定值，盗贼默认每回合回20点
   - 符文：死亡骑士符文每3回合冷却完成一个
3. **战斗节奏**：
   - 速度属性决定出手顺序，速度越快越先出手
   - 控制技能（昏迷、冰冻等）按回合数计算持续时间
   - 引导技能需要持续施法多个回合才能生效
