# 核心玩法设计

&gt; 经典回合制战斗，纯职业build玩法，还原魔兽世界WLK版本的核心体验

---

## 🎮 战斗系统设计

### 核心战斗机制
#### 回合制基础
- **出手顺序**：由角色的「急速」属性决定，速度越快越先行动
- **公共冷却（GCD）**：所有技能共享1回合的公共冷却时间
- **技能冷却**：每个技能有独立的冷却回合数
- **行动回合**：每个回合玩家可以选择：释放技能、使用物品、防御、更换目标

#### 资源系统
每个职业有独特的资源系统，还原WLK原版体验：

| 职业 | 资源系统 | 恢复机制 |
|------|---------|---------|
| 战士 | 怒气 | 受到伤害/造成伤害时获得，每回合自动衰减5点 |
| 圣骑士 | 圣能 | 某些技能触发获得，最多3点 |
| 死亡骑士 | 符文/符文能量 | 符文每3回合冷却1个，符文能量通过消耗符文获得 |
| 猎人 | 集中值 | 每回合恢复6点，上限100点 |
| 萨满祭司 | 法力值 | 每回合恢复基础法力值的2% |
| 盗贼 | 能量/连击点 | 能量每回合恢复20点，连击点通过特定技能获得 |
| 德鲁伊 | 法力值/能量/怒气 | 依形态而定 |
| 牧师 | 法力值 | 每回合恢复基础法力值的3% |
| 法师 | 法力值 | 每回合恢复基础法力值的2.5% |
| 术士 | 法力值/灵魂碎片 | 灵魂碎片通过特定技能获得，最多3片 |

#### 属性系统
完全还原WLK版本属性体系：

**基础属性**：
- 力量：增加物理攻击强度，板甲职业护甲加成
- 敏捷：增加物理暴击，攻击速度，盗贼/猫德AP
- 智力：增加法术强度，法术暴击，法力值上限
- 耐力：增加生命值上限
- 精神：增加法力回复速度，治疗效果加成

**战斗属性**：
- 命中：降低攻击被闪避/招架的概率
- 暴击：攻击/治疗暴击概率（基础1%）
- 急速：降低技能冷却时间，增加攻击速度
- 精通：各职业特有加成效果
- 护甲：降低受到的物理伤害
- 魔法抗性：降低受到的对应系魔法伤害

### 技能循环设计
每个职业有独特的技能循环，还原WLK版本的核心玩法：

#### 示例：战士（武器系）
- **起手**：冲锋 → 撕裂 → 致死打击
- **循环**：压制（可用时）→ 致死打击 → 斩杀（20%血以下）→ 英勇打击（泄怒）
- **冷却**：致死打击6回合，压制4回合，斩杀无冷却但需要20%血以下

#### 示例：法师（奥术系）
- **起手**：奥术冲击 × 4 → 奥术飞弹
- **循环**：奥术冲击叠层 → 触发飞弹速射时用奥术飞弹 → 蓝量低于30%时唤醒
- **冷却**：奥术飞弹10回合，唤醒30回合

---

### 控制与减益
#### 控制技能（按回合持续）
- 昏迷：目标无法行动，受到伤害增加50%
- 冰冻：目标无法行动，受到暴击增加100%
- 恐惧：目标随机行动，可能唤醒
- 沉默：目标无法施法
- 定身：目标无法移动，但可以施法

#### 减益效果
- 虚弱：降低目标伤害20%
- 破甲：降低目标护甲50%
- 诅咒：持续受到魔法伤害
- 中毒：持续受到自然伤害
- 疾病：持续受到伤害并降低治疗效果

## ⚔️ 职业与天赋系统

### 职业定位
10大职业，每个职业有3系天赋，覆盖所有职责：

| 职责 | 职业 |
|------|------|
| 坦克 | 战士（防护）、圣骑士（防护）、死亡骑士（鲜血）、德鲁伊（守护） |
| 治疗 | 圣骑士（神圣）、萨满（恢复）、德鲁伊（恢复）、牧师（戒律/神圣） |
| 物理DPS | 战士（武器/狂暴）、死亡骑士（冰霜/邪恶）、猎人（射击/生存）、盗贼（刺杀/战斗/敏锐）、德鲁伊（野性） |
| 法术DPS | 圣骑士（惩戒）、萨满（元素）、德鲁伊（平衡）、牧师（暗影）、法师（奥术/火焰/冰霜）、术士（痛苦/恶魔/毁灭） |

### 天赋系统
- 每系天赋树包含5层，每层1-3个可选天赋
- 底层核心天赋需要投入至少31点对应系天赋点才能点出
- 80级共71点天赋点
- 支持双天赋切换，可随时切换两套天赋配置

## 🎲 掉落与拾取

### 个人拾取
- 每个玩家独立ROLL点（1-100）
- 适合自己职业的装备掉率提高3倍
- 重复装备可以分解为材料

### ROLL点规则
- 需求（Need）：适合自己职业和天赋的装备
- 贪婪（Greed）：可以卖钱或分解的装备
- 分解（Disenchant）：直接分解为材料

### 套装掉落
- T1-T10套装完全还原
- 2件套/4件套效果强力
- 套装外观还原经典造型

## 📊 数值平衡设计

### 伤害公式
**物理伤害**：
```
最终伤害 = (武器伤害 + AP加成) × (1 - 护甲减伤) × (1 ± 暴击加成) × 命中判定
```

**法术伤害**：
```
最终伤害 = (基础伤害 + SP加成) × (1 - 抗性减伤) × (1 ± 暴击加成) × 命中判定
```

### 治疗公式
```
最终治疗 = (基础治疗 + SP加成) × (1 ± 暴击加成)
```

### 难度曲线
- 1-60级：平缓升级，体验经典内容
- 60-70级：外域内容，难度适中
- 70-80级：诺森德内容，终极挑战
