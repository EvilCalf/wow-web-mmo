# 开发 Roadmap

## 🎯 阶段一：最小可玩版本 (MVP) - 2周
**目标**：完成核心玩法闭环，可体验基础战斗和职业系统

### 功能清单
1. **账号系统**
   - 邮箱/手机号注册登录
   - 角色创建（联盟/部落选择，10大职业可选）
   - 角色基础属性展示

2. **职业系统**
   - 还原WLK版本10大职业核心技能
   - 基础天赋树（每个职业3系天赋，每系10个核心天赋）
   - 技能伤害公式和属性计算

3. **战斗系统**
   - 回合制战斗基础框架
   - PVE战斗（野怪战斗）
   - 自动战斗功能
   - 战斗奖励掉落

4. **装备系统**
   - 基础装备品质（白/绿/蓝/紫/橙）
   - 装备属性加成
   - 穿戴/卸下功能

5. **副本系统**
   - 首个副本：`怒焰裂谷`（1-10级）
   - 单人模式
   - 基础动态难度（随角色等级调整怪物属性）

### 交付标准
- 可创建角色，升级到10级
- 可通关怒焰裂谷副本
- 技能和天赋系统正常生效

---

## 🎯 阶段二：核心内容完善 - 4周
**目标**：完善核心玩法，加入社交和多人副本功能

### 功能清单
1. **副本系统**
   - 新增5个经典五人副本：`哀嚎洞穴`、`死亡矿井`、`影牙城堡`、`暴风城监狱`、`诺莫瑞根`
   - 多人组队功能（最多5人）
   - 完整动态难度系统（1人/3人/5人难度，普通/英雄难度）
   - 副本掉落匹配难度

2. **社交系统**
   - 好友系统（添加/删除/聊天）
   - 组队系统（创建队伍/加入队伍/队长权限）
   - 公会系统基础版（创建公会/加入公会/公会聊天）

3. **大地图系统**
   - 东部王国/卡利姆多大陆地图
   - 地图传送功能
   - 野怪分布和野外战斗
   - 飞行点解锁系统

4. **养成系统**
   - 专业技能（挖矿/采药/锻造/炼金等基础专业）
   - 坐骑系统（基础坐骑收集和使用）
   - 成就系统基础版

5. **交易系统**
   - 玩家交易功能
   - 拍卖行基础版

### 交付标准
- 可升级到60级
- 可组队通关所有五人副本
- 大地图可自由探索和传送

---

## 🎯 阶段三：高阶内容 - 6周
**目标**：加入团本、PVP和运营功能，达到公测标准

### 功能清单
1. **团本系统**
   - 首个团队副本：`纳克萨玛斯`（10人/25人模式）
   - 团队副本进度系统
   - BOSS机制还原
   - 套装掉落系统

2. **PVP系统**
   - 竞技场（1v1/2v2/3v3）
   - 战场（战歌峡谷/阿拉希盆地）
   - 荣誉系统和军衔
   - PVP装备兑换

3. **大地图玩法**
   - 世界BOSS刷新
   - 野外PVP区域
   - 日常任务系统
   - 节日活动基础框架

4. **运营系统**
   - 签到奖励
   - 商城系统（非数值道具）
   - 活动管理后台
   - 数据统计和分析

5. **性能优化**
   - 客户端性能优化（低端手机流畅运行）
   - 服务器负载优化（支持5000人同时在线）
   - 反作弊系统

### 交付标准
- 可升级到80级（WLK满级）
- 可通关纳克萨玛斯团本
- 支持5000人同时在线稳定运行

---

## 🎯 阶段四：持续迭代 - 长期
**目标**：持续更新内容，保持游戏活力

- 陆续开放更多经典副本（太阳井、冰冠堡垒等）
- 加入新职业和新种族
- 跨服功能（跨服副本/跨服竞技场）
- 移动端APP版本
- 更多玩法创新
