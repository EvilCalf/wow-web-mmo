# 魔兽世界图标资源

本目录包含魔兽世界Web MMO游戏所需的所有图标资源。

## 下载方法

### 方法一：使用Shell脚本（推荐）
```bash
cd shared
chmod +x download-icons.sh
./download-icons.sh
```

### 方法二：使用Node.js脚本
```bash
cd shared
node download-icons.js
```

## 图标来源

所有图标来自WowHead CDN，仅供学习交流使用。
- 官方网站: https://www.wowhead.com/
- CDN地址: https://wow.zamimg.com/

## 图标索引

查看 `index.json` 获取所有图标的映射关系。
